Model Predictive Control for Autonomous Vehicle

MATLAB/Simulink • MPC • Trajectory Tracking

Implemented Model Predictive Control (MPC) for autonomous vehicle trajectory tracking using MATLAB/Simulink. The controller enables precise following of reference paths while accounting for system dynamics and constraints.

The MPC approach predicts future vehicle behavior and optimizes control inputs, demonstrating superior accuracy compared to traditional PID controllers for complex trajectories.

Key Contributions:
  • Developed kinematic vehicle model with MPC controller
  • Compared MPC performance against PID for step-input and circular trajectories
  • Achieved accurate lateral position and yaw angle tracking
  • Implemented constraint handling for realistic vehicle control

Hand Exoskeleton for Rehabilitation

SolidWorks • MATLAB • Simscape • Genetic Algorithms

Designed and modeled a rehabilitation hand exoskeleton to restore finger movements in post-stroke patients. The project includes comprehensive kinematic and dynamic analysis, CAD design in SolidWorks, and simulation in MATLAB/Simscape.

Performed manipulability analysis and optimization using Genetic Algorithms to ensure safe human-robot interaction and optimal force transmission within natural hand range of motion.

Key Contributions:
  • Developed anatomically accurate human hand and exoskeleton models
  • Designed closed-chain mechanisms for MCP and PIP joints
  • Optimized link lengths using Genetic Algorithms based on manipulability
  • Ensured compatibility with natural hand anatomy and safety constraints

Control of Nonlinear Hand Exoskeleton

MATLAB • Adaptive Control • DMRAC • Nonlinear PID

Implemented and compared three control strategies for a hand exoskeleton: PD compensator multivariable control, inverse-dynamics nonlinear PID, and Direct Model Reference Adaptive Control (DMRAC).

Conducted comprehensive comparative analysis to evaluate performance, robustness, and efficiency of each control approach for rehabilitation applications.

Key Contributions:
  • Implemented three advanced control strategies for nonlinear system

Fuzzy Logic Control for Mobile Robot

MATLAB/Simulink • Fuzzy Logic • Mobile Robotics • Navigation

Designed and implemented fuzzy logic-based P, PI, PD, and PID-like controllers for unicycle mobile robot navigation. Used Mamdani inference systems to regulate linear velocity and steering rate for goal-reaching tasks.

PID-like fuzzy controller achieved the best performance with minimal error (~10⁻³) and almost no oscillations, demonstrating fuzzy logic as an effective alternative to classical PID for nonlinear systems.

Key Contributions:
  • Designed four fuzzy controllers emulating P, PI, PD, and PID behavior
  • Implemented complete unicycle kinematic model in Simulink
  • Conducted comparative evaluation of controller performance
  • Achieved superior results with PID-like fuzzy control

3D Hand Pose Estimation from Stereo Cameras

Python • OpenCV • PyTorch • Computer Vision • DLT

Implemented a 3D hand pose estimation pipeline using stereo cameras and Direct Linear Transformation (DLT). Supports multiple 2D keypoint detectors including Mediapipe, RTMPose, and LiteHRNet for triangulation.

Included fisheye camera distortion modeling to improve accuracy, achieving lower MPJPE error. The system is part of the MirrorGlove project for hand motion mirroring applications.

Key Contributions:
  • Implemented stereo camera triangulation using DLT
  • Integrated multiple 2D hand keypoint detectors
  • Modeled fisheye distortion for improved 3D accuracy
  • Developed filtering for reduced noise in 3D trajectories
